Class EvoSynth::Mutations::ExchangeMutation
In: lib/evosynth/operators/mutations/exchange_mutation.rb
Parent: Object

This mutations exchanges two genes in the genome of a given individual and returns a mutated individual. It is based on VERTAUSCHENDE-MUTATION (Weicker 2007, page 27) and DREIERTAUSCH-MUTATION (Weicker 2007, page 107)

This mutations does not destroy permutations.
