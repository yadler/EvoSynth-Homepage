Module EvoSynth::Mutations::Functions
In: lib/evosynth/operators/mutations/flip_functions.rb

Contains all predefined flip functions to be used with the BinaryMutation, EfficientBinaryMutation and OneGeneFlipping mutation.

Constants

FLIP_BOOLEAN = lambda { |gene| !gene }   inverts a boolean value
FLIP_BINARY = lambda { |gene| gene == 0 ? 1 : 0 }   inverts a integer in a boolean manner
FLIP_NEGATE_NUMBER = lambda { |gene| gene * -1 }   negates a integer
