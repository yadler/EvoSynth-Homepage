<h2>What is EvoSynth?</h2> <p>EvoSynth is a framework (written in ruby) for rapid development and prototyping of evolutionary algorithms. It is designed with <i>simplicity and productivity</i> as main goals, so don’t expect it to compete with the performance of the established frameworks.</p> <h2>Latest news</h2> <h3>09th October 2010: Website ported to nanoc</h3> <p>I just ported to website of EvoSynth to nanoc. This should make changes more pleasant in the future.</p> <h3>26th July 2010: New Website</h3> <p>While EvoSynth 0.2 is still in development, EvoSynth finally got a real Website. It replaced the rdoc source code documentation, which is now located <a href="/rdocs/rdoc01/">here</a>.</p> <h3>23th July 2010: Ohloh project (Metrics for EvoSynth)</h3> <p>Today, I found the time to setup a <a href="http://www.ohloh.net/p/evosynth">Ohloh project page</a> for EvoSynth. Even if Ohloh says that EvoSynth has <a href="http://www.ohloh.net/p/evosynth/factoids/3489257">extremely well-commented source code</a> there is still a lot of space for improvements.</p> <h3>12th March 2010: EvoSynth 0.1 released</h3> <p>I just made the first release (0.1) of my thesis project called “EvoSynth“, which stands for Evolutionary Computation Synthesizer. Read the complete <a href="http://yadler.wordpress.com/2010/03/12/evosynth-0-1-released/">release announcement</a>.</p>
